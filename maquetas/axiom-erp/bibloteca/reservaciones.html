Políticas de Reservación

Duración Máxima de Reserva
Las reservaciones son válidas por 7 días. Si no se recoge dentro de este plazo, la reservación se cancela automáticamente.
Límite de Reservaciones
Un usuario puede tener hasta 5 reservaciones activas simultáneamente para evitar monopolizar recursos.
Cola de Espera
Las reservaciones se procesan en orden FIFO (primero en entrar, primero en salir) por orden de prioridad.
Notificaciones Automáticas
Se envían notificaciones por email cuando el libro solicitado está disponible. El usuario tiene 48 horas para recogerlo.
Cancelación de Reservas
Los usuarios pueden cancelar sus reservaciones en cualquier momento. Las cancelaciones abren el espacio para otros usuarios.
Renovación de Préstamo
Si hay reservaciones pendientes para un libro, el préstamo actual no puede renovarse. Debe devolverse puntualmente.
